COMO PUBLICAR RÁPIDO
NETLIFY:
- Entre no Netlify.
- Arraste esta pasta inteira ou o ZIP para a área de deploy.
- Depois conecte seu domínio.

GITHUB PAGES:
- Envie os arquivos para um repositório.
- Ative Pages nas configurações.
- Como é um site estático, não precisa de Python/Flask.

META ADS
- Evite antes/depois, promessas de perda de peso, linguagem que presume atributo pessoal
  ("Você está acima do peso?") e resultados garantidos.
- Use uma linguagem neutra sobre hábitos, rotina e bem-estar.
- O banner incluído foi gerado como material inicial e contém textos promocionais fortes.
  Recomenda-se trocar por uma versão revisada, sem alegações não comprovadas, antes de anunciar.
